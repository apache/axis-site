Welcome to Apache Rampart

Apache Rampart™ provides implementations of the WS-Sec* specifications for Apache Axis2, based on Apache WSS4J and the Apache Axiom DOOM implementation. Rampart implements the following specifications:

  • WS - Security 1.0
  • WS - Security 1.1
  • WS - Secure Conversation - February 2005
  • WS - Security Policy - 1.1 - July 2005
  • WS - Security Policy - 1.2
  • WS - Trust - February 2005
  • WS - Trust - WS-SX specification
  • SAML Specification - 1.1
  • SAML Specification - 2.0
