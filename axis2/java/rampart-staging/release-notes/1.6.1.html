Apache Rampart 1.6.1 Release Note

The Apache Rampart 1.6.1 release updates WSS4J to version 1.5.12 and ensures compatibility with Axis2 1.6.1. Please note that Rampart 1.6.1 will not work with Axis2 1.6.0 and that users of Axis2 1.6.1 are required to update to Rampart 1.6.1, i.e. Rampart 1.6.0 doesn’t work with Axis2 1.6.1.
