Since we use the same env->error from Axis2 we need to start from a AXIS2_ERROR_LAST + some number to prevent ambiguity

enum sandesha2_error_codes

Enumerator:
SANDESHA2_ERROR_PENDING_HEADER_MUST_HAVE_ATTRIBUTE_PENDING  The following has to be the last error value all the time. All other error codes should appear above this. SANDESHA_ERROR_LAST is used to track the number of error codes present for the purpose of sizing the error message array.
