public class AxisServerMBeanExporter
extends Object
implements ServletContextAttributeListener

Listener that registers the MBeans for the AxisServer created by AxisServlet. To enable MBean registration in your Web application, add the following configuration to web.xml: 

 <listener>
     <listener-class>org.apache.axis.management.servlet.AxisServerMBeanExporter</listener-class>
 </listener>
