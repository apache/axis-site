Plugin Documentation

Goals available for this plugin:

GoalDescription
axis-server:start-daemonStart a daemon.
axis-server:start-serverStart a StandaloneAxisServer instance in a separate JVM.
axis-server:start-webappNo description.
axis-server:stop-allStop all processes created by StartServerMojo.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven2.0
JDK1.4
MemoryNo minimum requirement.
Disk SpaceNo minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.apache.axis</groupId>
          <artifactId>maven-axis-server-plugin</artifactId>
          <version>1.4.1-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.apache.axis</groupId>
        <artifactId>maven-axis-server-plugin</artifactId>
        <version>1.4.1-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
