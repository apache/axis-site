WebServices - Axis

Axis 日本語メーリングリスト

下記の注意点を読んで確実に理解してから、メーリングリストに参加してください。

利用上の注意

  • 件名(Subject)は、半角英数字のみとしてください。

    • 　現在、スパム対策として、日本語の件名を利用できないように設定しています。勿論、本文は日本語で大丈夫です。

  • メーリングリストに HTML メールを送らないでください。

    • 　あなたが Outlook/Outlook Express/Eudora といったメールソフトを利用していいる場合、HTML メールを送っている可能性があります。テキスト形式(プレーンテキスト)でメールを送るように設定してください。

参加方法

中程度の流通量 [参加] [退会] [参加(ダイジェスト)] [退会(ダイジェスト)] [メーリングリストにメールを送る]

このメーリングリストは、日本語でAxis の使用方法等について質問をしたい、知識を共有したい、問題を議論したい開発者のためのものです。もし、英語のメーリングリストに参加したい場合は、こちらをご覧下さい。

【参加手順】

  1. 参加申込みメールの送信

    2. axis-user-ja-subscribe@ws.apache.orgに空のメールを送ります。

  2. 申請受付メールの受信

    3. 件名が「confirm subscribe to axis-user-ja@ws.apache.org」という、下記のようなメールを受領します。 
    From: axis-user-ja-help@ws.apache.org
Sent: Wednesday, November 24, 2004 2:17 PM
To: toshi@apache.org
Subject: confirm subscribe to axis-user-ja@ws.apache.org

Hi! This is the ezmlm program. I'm managing the
axis-user-ja@ws.apache.org mailing list.

To confirm that you would like

   toshi@apache.org

added to the axis-user-ja mailing list, please send
an empty reply to this address:

   axis-user-ja-sc.1101273444.oieeppnodmhnilnnfhmg-toshi=apache.org@ws.apache.org

Usually, this happens when you just hit the "reply" button.
If this does not work, simply copy the address and paste it into
the "To:" field of a new message.
               :

  3. 意思確認メールの送信

    4. 「返信ボタン」を利用し、上記メールに対して空のメールを返信して下さい。

  4. 登録完了メールの受信

    5. メーリングリストに登録が完了すると、下記のメールが送られてきます。 
    From: axis-user-ja-help@ws.apache.org
[mailto:axis-user-ja-help@ws.apache.org]
Sent: Wednesday, November 24, 2004 2:18 PM
To: toshi@apache.org
Subject: WELCOME to axis-user-ja@ws.apache.org

Hi! This is the ezmlm program. I'm managing the
axis-user-ja@ws.apache.org mailing list.

Acknowledgment: I have added the address

   toshi@apache.org

to the axis-user-ja mailing list.

Welcome to axis-user-ja@ws.apache.org!

Please save this message so that you know the address you are
subscribed under, in case you later want to unsubscribe or change your
subscription address.
                  :

    以上で、メーリングリストへの参加手続きは終了です。
